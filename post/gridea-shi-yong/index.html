之前用Hexo，不过命令行用着很不习惯，Gridea客户端应该还不错，如果不考虑蛋疼的同步...

👇记录一下Gridea的操作

同步

0.9.3版本加入了代理，但也无法同步
解决方法：配合Github Desktop

1️⃣ 在Gridea设置中找到所在路径，删除output文件夹
2️⃣ Github Desktop中找到github上的.github.io仓库，链接到本地的路径+output文件夹中
3️⃣ 每次在Gridea同步后（失败没有关系）在Github Desktop上push即可💁

ISSUES：
1️⃣ Github Desktop显示Authentication Failed
💡 Repository - Repository settings 修改remote为远程仓库的https链接
