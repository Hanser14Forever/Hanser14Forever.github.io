7.1

1.两数之和

标签：哈希表

思路：创建哈希表，对每个x，先查询表中是否存在target-x，没有则将x插入到哈希表中。

时间复杂度：O(N)

空间复杂度：O(N)

Python

class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        ht = dict()
        for i in range(len(nums)):
            if target-nums[i] in ht.keys():
                return [ht[target-nums[i]], i]
            ht[nums[i]] = i
  • Python中哈希表：用字典dict()实现

C++

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map<int, int> ht;
        for(int i=0; i<nums.size(); ++i){
            auto it = ht.find(target-nums[i]);
            if(it!=ht.end()){
                return {it->second, i};
            }
            ht[nums[i]] = i;
        }
        return {};
    }
};
  • C++中哈希表：用unordered_map<type, type>实现
    • ele->first指代键；ele->second指代值
  • C++中在哈希表中寻找元素：
    • 先创建一个迭代器：auto it = hashtable.find(ele)
    • 表中包含该元素：it!=hashtable.end()
  • C++中返回列表：用{}，与Python区别开

2.两数相加

标签：链表

思路：模拟，设置好进位，考虑好最后一个节点

时间复杂度：O(max(m,n))

空间复杂度：O(1)

Python

class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode()
        carry, curr = 0, dummy
        while l1 or l2 or carry:
            s = (l1.val if l1 else 0) + (l2.val if l2 else 0) + carry
            carry = s // 10
            s %= 10
            curr.next = ListNode(s)
            curr = curr.next
            l1 = l1.next if l1 else None
            l2 = l2.next if l2 else None
        return dummy.next
  • Python中新建空链表：l = ListNode()
  • Python中新建值为val的链表：l = ListNode(val)
  • Python中，变量当满足条件cond时赋值a，否则赋值b：v = a if cond else b

C++

class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode* dummy = new ListNode();
        int carry = 0;
        ListNode* curr = dummy;
        while(l1 || l2 || carry){
            int s = (l1 ? l1->val : 0) + (l2 ? l2->val : 0) + carry;
            carry = s / 10;
            curr->next = new ListNode(s % 10);
            curr = curr->next;

            l1 = l1 ? l1->next : nullptr;
            l2 = l2 ? l2->next : nullptr;
        } 
        return dummy->next;
    }
};
  • C++中新建空链表：ListNode* l = new ListNode()
  • C++中新建值为val的链表：ListNode* l = new ListNode(val)
  • C++中，变量当满足条件cond时赋值a，否则赋值b：v = cond ? a : b

445.两数相加II

标签：链表，栈

思路：将两链表的值压入栈中，依次取出相加

时间复杂度：O(max(m,n))

空间复杂度：O(m+n)

Python

class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        s1, s2 = [], []
        while l1:
            s1.append(l1.val)
            l1 = l1.next
        while l2:
            s2.append(l2.val)
            l2 = l2.next
        
        head, carry = None, 0
        while s1 or s2 or carry:
            s = (s1.pop() if s1 else 0) + (s2.pop() if s2 else 0) + carry
            carry = s // 10
            curnode = ListNode(s%10)
            curnode.next = head
            head = curnode
        
        return head

  • Python中栈：由列表list()实现

    • 栈的push：l.append(ele)

    • 栈的pop：l.pop()

    • 栈顶元素：l[-1]

  • 板子：Python链表中，每次新加节点到表头，实现方法：

    head = None
while cond:
    curnode = ListNode(val)
    curnode.next = head
    head = curnode

C++

class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        stack<int> s1, s2;
        while (l1) {
            s1.push(l1->val);
            l1 = l1->next;
        }
        while (l2) {
            s2.push(l2->val);
            l2 = l2->next;
        }

        int carry = 0;
        ListNode* head = nullptr;
        while (!s1.empty() || !s2.empty() || carry) {
            int s = (!s1.empty() ? s1.top() : 0) + (!s2.empty() ? s2.top() : 0) + carry;
            if (!s1.empty()) s1.pop();
            if (!s2.empty()) s2.pop();
            carry = s / 10;
            ListNode* curnode = new ListNode(s%10);
            curnode->next = head;
            head = curnode;
        }
        return head;
    }
};

  • C++中栈的实现：stack<int> s

    • 栈的push：s.push(ele)

    • 栈的pop：s.pop()

    • 栈顶元素：s.top()

  • 板子：C++链表中，每次新加节点到表头，实现方法：

    ListNode* head = nullptr;
while (cond) {
	ListNode* curnode = new ListNode(val);
    curnode->next = head;
    head = curnode;
}
